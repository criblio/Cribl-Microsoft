# Deprecated components

Moved here on 2026-07-13. These components are superseded and receive no
further development. Full history is preserved (moved with `git mv`; use
`git log --follow <file>` to trace it), and the pre-move layout is tagged
`pre-deprecation-2026-07-13`.

| Component | What it was | Superseded by |
| --- | --- | --- |
| `Azure/` | PowerShell DCR automation, discovery tools, labs | `soc-optimizationtoolkit/` (DCR Automation screen, Event Hub Discovery, deploy flows) |
| `Cribl-Microsoft_IntegrationSolution/` | Electron/React integration GUI | `soc-optimizationtoolkit/` (Cribl.Cloud app + local app) |
| `Lookups/` | Standalone lookup assets | Pack-embedded lookups built by the toolkit's pack assembly |
| `packs/` | Hand-maintained Cribl packs | Packs generated by the toolkit (Pack Maintenance screen) |
| `SOC-OptimizationToolkit_v1/` | First-generation toolkit | `soc-optimizationtoolkit/` (v2 rewrite) |
| `Start-App-Windows.bat` / `Start-App-macOS.sh` | Launchers for the Electron GUI | `soc-optimizationtoolkit/` app (Cribl.Cloud upload or `npm run dev` in `apps/local-app`) |

Do not add new work here. If something is still needed, port it into
`soc-optimizationtoolkit/` instead.
