## Configuration File Locations

### Updated Paths
All configuration files are now located in the `core/` directory:
- `core/azure-parameters.json` - Azure resource configuration
- `core/operation-parameters.json` - Script behavior settings
- `core/NativeTableList.json` - Native Azure tables to process
- `core/CustomTableList.json` - Custom tables to process
- `core/custom-table-schemas/` - Custom table schema definitions

### Output Locations
Generated files are organized in the `core/` directory:
- `core/generated-templates/` - ARM templates for manual deployment
- `core/cribl-dcr-configs/` - Cribl Stream configuration exports
